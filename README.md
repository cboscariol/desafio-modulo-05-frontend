### Rodando o projeto

Na pasta raiz do projeto, você pode rodar:

#### `npm install`

e então:

#### `npm start`

Uma aba será aberta localmente no seu browser (http://localhost:3000) para rodar as duas ao mesmo tempo, rode em duas portas distintas (http://localhost:3001)
